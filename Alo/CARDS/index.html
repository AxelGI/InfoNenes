Mapa de alertamiento por temperaturas altas

Este proyecto presenta un mapa interactivo de los estados de México afectados por altas temperaturas, desarrollado con Python y la librería Folium. De esta manera, el proyecto combina el análisis de datos con herramientas de visualización geoespacial para facilitar la comprensión de las temperaturas extremas en el país.
